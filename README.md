# Features

- User Signup & Login
- JWT Authentication
- Role-Based Access Control (Admin/Member)
- Project Management
- Task Assignment & Status Tracking
- Dashboard Analytics
- Team Management
- Protected Routes
- Responsive UI

---

# Admin Features

- Create Projects
- Create & Assign Tasks
- View All Tasks
- Delete Tasks
- Manage Team Members

---

# Member Features

- View Assigned Tasks
- Update Task Status
- Access Personal Dashboard

# Installation

## Backend

```bash
cd backend
npm install
npm run dev
```

Create `.env`:

```env
MONGO_URI=mongodb+srv://sushantkhairnar27_db_user:<password>@teamtask.lakdv7n.mongodb.net/teamtaskmanager?retryWrites=true&w=majority
JWT_SECRET=your_jwt_secret_key_here
```

---

## Frontend

```bash
cd frontend
npm install
npm run dev
```

Create `.env`:

```env
VITE_API_URL=http://localhost:5000/api
```
